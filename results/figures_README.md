# Results

This folder contains figures generated from the MED-Cli longitudinal dataset
(1,249 internal medicine hospitalisations, IRCCS San Raffaele Hospital).

Patient-level data is not included. All figures are aggregate-level visualisations
that do not contain patient-identifying information.

---

## Figures

| Figure | Description |
|--------|-------------|
| `followup_distribution.png` | Distribution of follow-up duration across hospitalisations |
| `observation_density.png` | Laboratory observations per patient; ~42% of patients have fewer than 10 observations |
| `selected_biomarker_trajectories.png` | Example longitudinal trajectories for selected biomarkers |
| `missingness_heatmap.png` | Structured missingness across biomarkers and time — non-random, clinically patterned |
| `correlation_day0_heatmap.png` | Baseline (Day 0) biomarker correlation structure |
